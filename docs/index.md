# Gryia Project Guide

This documentation is the hub for the Gryia game. Use the sections above to jump into the core gameplay concepts or the lore bible.

## Getting Started

1. Run `mkdocs serve` to preview locally.
2. Open the **Game** section to review mechanics and progression plans.
3. Explore **Lore** for factions, locations, and narrative hooks.

## Structure

- `Game` — design pillars, systems, and MVP tasks.
- `Lore` — world overview, factions, locations, and story threads.
- `Home` — this landing page with quick instructions.
